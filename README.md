# HTML5 WebSocket Chat

Chat escrito com Node.js, Socket.io, Webpack, Babel-ES6 e Flexbox. :)

## Como iniciar o servidor?

- Execute os comandos abaixos para instalar, construir e iniciar o servidor.

```bash
$ npm install
$ npm run build
$ npm start
```
